Exemples de héros pour vos aventures

Comment créer votre personnage

  1. Prenez une fiche personnage vierge (PDF) (Dernière version le 10/03/2026).
  2. Choisissez un nom et une classe : Sélectionnez une classe (se référer au tableau des classes) qui vous donnera votre DEF de base, votre vie (à calculer avec un lancer de dé) ainsi que les armes / sorts / armures et boucliers que vous pouvez utiliser.
  3. Notez vos caractéristiques : Noter vos PV, votre bonus de DEF de base, ainsi que les armes / sorts / armures et boucliers que vous souhaitez utiliser.
    Notez bien qu'un personnage ne peut porter que 3 slots d'armes, et jusqu'à 5 sorts (5 pour les Tisseurs Arcanique et Marche-Rêve, 2 pour les Lame-Litanie).
  4. Détaillez vos armes et sorts : Indiquez toutes les précisions de vos armes et de vos sorts sur votre fiche personnage (portée, dégâts, tours de récupérations, …)
  5. Définissez vos bonus d'attributs : 2 choix : Distribuer +3, +2, +1, +0, -1, OU +2, +2, +1, +0, +0 pour chacun des attributs (exemple : PUI = +2, AGI = +2, VAI = +1, INS = +0, INT = +0).
  6. Écrivez le lore de votre personnage : Si vous le souhaitez, écrivez un mini lore de votre personnage, d'où vient-il, ce qu'il faisait par le passé, ce qu'il fait là, etc…
