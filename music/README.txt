===========================================
INSTRUCTIONS POUR LA MUSIQUE DE FOND
===========================================

1. Téléchargez votre fichier audio au format MP3
2. Renommez-le en "background.mp3"
3. Placez-le dans ce dossier (music/)

Le système de musique est déjà intégré dans le site.
Une fois le fichier ajouté, le bouton de contrôle de la musique
apparaîtra en bas à droite de chaque page.

Note : Le fichier doit être nommé exactement "background.mp3"
pour que le système fonctionne correctement.
